How to Use This App

  1. Enter one or more Substack RSS feed URLs (one per line) in the first box. Example: https://mmacfadden.substack.com/feed
  2. Optionally, enter specific article URLs (one per line) in the second box to load individual articles.
  3. Click Load Articles to fetch and display articles.
  4. Use the gear icon to set how many articles to load per feed.
  5. Check the printer icon to enable print-friendly layout.
  6. Click the file icon to return to the feed input panel.
  7. Click the question mark icon (this panel) to view instructions.

Note: This app works best on desktop browsers.
